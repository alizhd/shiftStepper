Local implementation of per timer tick behavior. 

This function manages the per-\/tick behavior of a block of switches. It is expected to be called from the doBoardTick() function of the board that contains this device.

This function writes the switches specified in the channels array with the states specified in the bit string written with \hyperlink{classshift_switch_block_abc0f22095833ef9f9d84ae8b29934357}{setSwitches()}.
